# 🚀 AI Prompt Improver with Cloud Integration

## 📌 Overview
This project improves user prompts using rule-based NLP techniques and enhances them using AI. The final output is stored locally and uploaded to AWS S3 for cloud storage.

---

## ⚙️ Features
- ✨ Prompt enhancement using Python logic
- 🤖 AI-based refinement (OpenAI API)
- 🔁 Fallback system if AI fails
- 📁 Output saved to file
- ☁️ Cloud upload using AWS S3

---

## 🛠️ Technologies Used
- Python
- OpenAI API
- AWS S3 (boto3)

---

## 🚀 How It Works
1. User enters a prompt  
2. System improves it using rule-based logic  
3. AI further refines the prompt  
4. Output is saved locally  
5. File is uploaded to AWS S3
